##  Project Pipeline (Notebook Order)

**Cleaning.ipynb**  
Data cleaning: handling missing values, formatting issues, duplicates, and inconsistencies.

 **Merging_v2.ipynb**  
Merging multiple data sources to create a final, usable dataset.

 **01_Outlier_Detection.ipynb**  
Detection and analysis of outliers that could negatively impact model performance.

 **02_Correlation.ipynb**  
Correlation analysis to identify relevant features and reduce redundancy.

 **03_Model_Baseline.ipynb**  
Creation of a baseline model used as a reference for performance comparison.

 **04_Model_Comparison.ipynb**  
Comparison of multiple machine learning algorithms to identify the best-performing model.

 **05_PCA_experiments.ipynb**  
Experiments using **PCA** for dimensionality reduction and
evaluation of its impact on model performance.

 **06_hyperparameter_tuning.ipynb**  
Hyperparameter optimization of the selected model to further improve performance.
